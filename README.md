# Modul 13 - Git dan GitHub

Proyek ini digunakan untuk latihan dasar kontrol versi menggunakan Git dan GitHub,
sekaligus sebagai persiapan pengembangan mini project website interaktif pada
mata kuliah Pemrograman Web 1. Proyek berisi halaman web sederhana yang
menampilkan deskripsi interaktif dan disimpan riwayat perubahannya menggunakan
Git, lalu dihubungkan ke repository remote di GitHub.

## Deskripsi Proyek
Halaman web ini menampilkan judul, deskripsi singkat, dan tombol interaktif.
Saat tombol ditekan, teks deskripsi akan berubah dan dapat dikembalikan lagi
ke teks semula (toggle), sekaligus mencatat jumlah klik pada console browser.

## Fitur
- Struktur HTML sederhana dengan elemen judul, deskripsi, dan footer
- Styling CSS dengan efek bayangan, transisi halus, dan tampilan kartu modern
- Interaksi JavaScript untuk mengubah dan mengembalikan teks deskripsi (toggle)
- Penghitung jumlah klik tombol
- Riwayat perubahan proyek tercatat melalui commit Git yang terstruktur
- Pengembangan fitur menggunakan branch terpisah (fitur-deskripsi, fitur-tampilan)

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Git dan GitHub untuk kontrol versi

## Cara Menjalankan Proyek
1. Clone atau unduh repository ini.
2. Buka file `index.html` menggunakan browser.
3. Klik tombol "Ubah Deskripsi" untuk melihat interaksi JavaScript.
